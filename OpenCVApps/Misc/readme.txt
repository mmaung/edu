Combine left and right images to combined images

images will be read from LeftImages directory with named 0.png, 1.png, 2.png,..., x.png, and
RightImages directory also with files named 0.png, 1.png, 2.png, ...., x.png

output images will be in CombinedImages directory with files named 0.png, 1.png, 2, png, ...., x.png

to invoke the function: ./combinedImages x
